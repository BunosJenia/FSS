Files Sharing System позволит пользователям оставлять на хранения свои файлы.

Права:
Администратор - Имеет право выполнять любые операции без каких бы то ни было ограничений.
Супер-пользователь - те же права что и у пользователя, но размер хранилища больше(нет ограничения по размеру файлов).
Пользователь - имеет свое хранилище(определенного размера) в которое может загружать файлы(есть ограничение на
размер, общий или по формату файла), скачку файлов из хранилища, передача ссылки на скачку файла любому лицу,
делать приватными файлы(скачивать может только сам пользователь).
Гость - может скачать файл по указанной ему ссылке(от пользователя), а так же может зарегистрироваться.


Администраторы могут:
- изменять допустимый размер загрузок для файлов, каталогом и т.д.;
- давать более высокие права пользователям;
- банить/разбанивать пользователей;

На данном этапе реализованно:
- каркас архитектуры.
